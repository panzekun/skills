# Task 02: 数据层准备

## 任务目标
从 `.claude/skills/` 目录提取技能数据，创建统一的数据配置文件。

## 具体步骤

### 1. 定义数据结构
- [ ] 在 `src/types/skill.ts` 中定义 Skill 接口:
  ```typescript
  interface Skill {
    id: string
    name: string
    nameEn: string
    description: string
    icon: string
    category: string
    scenarios: string[]
    features: string[]
    examples: string[]
    usageGuide: string
  }
  ```

### 2. 创建技能数据文件
- [ ] `src/data/skills.ts` - 手动提取 7 个技能的核心数据
- [ ] 从以下文件提取数据:
  - task-breakdown
  - ui-component-generator
  - performance-auditor
  - page-scaffold
  - debug-fixer
  - api-integration-helper
  - pr-writer

### 3. 创建数据工具函数
- [ ] `src/utils/skillHelper.ts` - 技能数据查询工具
  - `getAllSkills()` - 获取所有技能
  - `getSkillById(id)` - 根据 ID 获取技能
  - `getSkillsByCategory(category)` - 按分类获取技能

## 验收标准
- [ ] 7 个技能数据完整录入
- [ ] 数据结构符合 TypeScript 类型定义
- [ ] 工具函数可以正常查询数据
- [ ] 数据包含: 名称、描述、使用场景、核心功能、示例

## 相关文件
- `src/types/skill.ts`
- `src/data/skills.ts`
- `src/utils/skillHelper.ts`
- `.claude/skills/*/SKILL.md` (数据源)
