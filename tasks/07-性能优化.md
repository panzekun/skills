# Task 07: 性能优化 (可选)

## 任务目标
优化应用性能，减少 bundle 体积，提升加载速度。

## 具体步骤

### 1. 路由懒加载
- [ ] 使用 React.lazy 和 Suspense
- [ ] 懒加载页面组件:
  ```typescript
  const HomePage = lazy(() => import('./pages/HomePage'))
  const SkillDetailPage = lazy(() => import('./pages/SkillDetailPage'))
  ```

### 2. 组件优化
- [ ] 使用 React.memo 包裹纯展示组件
- [ ] 使用 useMemo 缓存计算结果
- [ ] 使用 useCallback 缓存回调函数

### 3. Ant Design 按需加载
- [ ] 检查是否已启用按需加载
- [ ] 只导入使用的组件

### 4. 代码分割
- [ ] 分析 bundle 体积
- [ ] 拆分大型依赖(如代码高亮库)

### 5. 图片优化
- [ ] 使用 WebP 格式(如有图片)
- [ ] 添加懒加载
- [ ] 压缩图片

### 6. 缓存策略
- [ ] 配置 Vite build 缓存
- [ ] 添加 Service Worker(可选)

## 验收标准
- [ ] 首屏加载时间 < 2s
- [ ] Bundle 体积合理
- [ ] Lighthouse 性能评分 > 90
- [ ] 路由切换流畅

## 相关文件
- `src/router/index.tsx`
- `src/pages/`
- `vite.config.ts`

## 注意事项
此任务为可选优化项，在基础功能完成后再进行。
