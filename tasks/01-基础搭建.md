# Task 01: 基础搭建

## 任务目标
搭建项目基础架构，包括路由配置、页面骨架、Layout 组件和 TypeScript 类型定义。

## 具体步骤

### 1. 安装依赖
```bash
npm install react-router-dom
npm install -D @types/node
```

### 2. 配置路由结构
- [ ] 在 `src/main.tsx` 中引入 BrowserRouter
- [ ] 创建 `src/router/index.tsx` - 路由配置文件
- [ ] 定义路由:
  - `/` - 首页
  - `/skills/:skillId` - 技能详情页

### 3. 创建页面骨架
- [ ] `src/pages/HomePage/index.tsx` - 首页
- [ ] `src/pages/SkillDetailPage/index.tsx` - 技能详情页
- [ ] `src/pages/NotFoundPage/index.tsx` - 404 页面

### 4. 创建 Layout 组件
- [ ] `src/components/Layout/index.tsx` - 主布局
- [ ] `src/components/Layout/Header.tsx` - 顶部导航
- [ ] `src/components/Layout/Footer.tsx` - 底部信息

### 5. TypeScript 类型定义
- [ ] `src/types/skill.ts` - 技能数据类型

## 验收标准
- [x] 依赖安装成功
- [ ] 路由可以正常切换
- [ ] 访问 `/` 显示首页骨架
- [ ] 访问 `/skills/task-breakdown` 显示详情页骨架
- [ ] 访问不存在的路由显示 404 页面
- [ ] Layout 组件正常渲染(Header + Content + Footer)
