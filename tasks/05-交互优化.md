# Task 05: 交互优化

## 任务目标
优化用户交互体验，添加动画、过渡效果、响应式优化。

## 具体步骤

### 1. 页面过渡动画
- [ ] 使用 CSS transition 或 Ant Design motion
- [ ] 路由切换时添加淡入淡出效果

### 2. 卡片 hover 效果
- [ ] SkillCard hover 时:
  - 阴影加深
  - 轻微上浮(transform: translateY)
  - 边框高亮

### 3. 响应式布局优化
- [ ] 测试移动端(375px)
- [ ] 测试平板(768px)
- [ ] 测试桌面(1200px+)
- [ ] 调整间距、字体大小

### 4. Loading 状态
- [ ] 页面加载时显示 Ant Design Spin
- [ ] 数据加载时的骨架屏(可选)

### 5. 滚动优化
- [ ] 添加平滑滚动
- [ ] 返回顶部按钮(Ant Design BackTop)

### 6. 交互反馈
- [ ] 按钮点击反馈
- [ ] 链接 hover 效果
- [ ] 表单交互(如有)

## 验收标准
- [ ] 页面切换流畅
- [ ] 卡片 hover 效果自然
- [ ] 移动端布局正常，无横向滚动
- [ ] Loading 状态正常显示
- [ ] 返回顶部按钮在长页面中出现
